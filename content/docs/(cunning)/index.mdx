<Lang lang="en">

**Cunning3D** is a procedural modeling ecosystem built from scratch for **high performance, cross-platform deployment, and modern workflows**.

It is more than just software; it is a **complete geometry computing solution**. Written in Rust, it spans the entire pipeline from desktop DCC to game engine integration (Unity) and web preview (WASM).

### 1. Cunning3D (Editor)
Currently in rapid development, building from source is recommended:
```bash
git clone https://github.com/YourRepo/Cunning3D.git
cd Cunning3D
cargo run --release
```

### 2. Cunning Engine (Unity Plugin)
1. Copy library files from `crates/cunning_core_ffi/dll` to your Unity project's `Assets/Plugins` folder.
2. Install the corresponding C# scripts.
3. See [Unity Integration](/docs/platforms/unity) for details.
