# File Node

The File node is the primary entry point for importing external geometric data into Cunning3D. It features a unified interface that automatically detects and handles various standard 3D file formats through an extensible registry system.

<Drawio file="content/docs/diagrams/file-node.drawio" />

## Supported Formats

The node currently supports the following industry-standard formats:

| Format | Extensions | Description | Supported Features |
| :--- | :--- | :--- | :--- |
| **FBX** | `.fbx` | Autodesk Filmbox | Geometry, Normals, UVs, Colors |
| **OBJ** | `.obj` | Wavefront Object | Geometry, Normals, UVs |
| **GLTF** | `.gltf`, `.glb` | GL Transmission Format | Full PBR Scene support (flattened to geometry) |
| **STL** | `.stl` | Stereolithography | Geometry only (Triangle Soup) |
| **PLY** | `.ply` | Polygon File Format | Point Clouds, Scanned Data |

## Parameters

| Parameter | Type | Description |
| :--- | :--- | :--- |
| **File** | `FilePath` | The absolute or relative path to the source file. The file picker automatically filters based on supported extensions. |
