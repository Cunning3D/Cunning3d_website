## Node Anatomy

<Lang lang="en">
Every node has:
</Lang>
<Lang lang="zh">
每个节点通常包含：
</Lang>

| Part | Description |
|------|-------------|
| **Inputs** | Geometry connections (0-N) |
| **Outputs** | Result geometry (usually 1) |
| **Parameters** | Controls (sliders, toggles, etc.) |

## Common Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| Group | String | Target element group |
| Class | Enum | Point/Vertex/Prim/Detail |
| Keep Original | Bool | Preserve input geometry |

## Editor vs Runtime

<Lang lang="en">
Nodes are tagged as:
</Lang>
<Lang lang="zh">
节点可能带有以下标签：
</Lang>

| Tag | Description |
|-----|-------------|
| `runtime` | Available in shipped builds |
| `editor` | Cunning3D editor only |

<Callout type="warning">
<Lang lang="en">
CDA files using editor-only nodes will fail to load in runtime builds.
</Lang>
<Lang lang="zh">
如果 CDA 文件包含 editor-only 节点，在 runtime 构建中会加载失败。
</Lang>
</Callout>

## Quick Reference

### Generators

| Node | Description |
|------|-------------|
| Box | Axis-aligned box with divisions |
| Sphere | UV sphere with rings/segments |
| Grid | Flat grid with divisions |
| Circle | N-sided polygon or polyline |
| Tube | Cylinder/cone with caps |
| Torus | Donut shape |

### Modeling

| Node | Description |
|------|-------------|
| PolyExtrude | Extrude faces with inset |
| PolyBevel | Chamfer edges/vertices |
| Boolean | Union/Intersect/Subtract |
| Bridge | Connect edge loops |
| Subdivide | Catmull-Clark subdivision |

### Transform

| Node | Description |
|------|-------------|
| Transform | Translate/Rotate/Scale |
| Copy | Instance geometry to points |
| Mirror | Reflect across plane |
| Scatter | Random point distribution |
