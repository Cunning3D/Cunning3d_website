## Integration Patterns

### Direct FFI

For Unity and other C-compatible engines:

```csharp
[DllImport("cunning_core_ffi")]
public static extern ulong cunning_geo_create_cube(float size, int dx, int dy, int dz);
```

### Native Crate

For Rust engines like Bevy:

```rust
use cunning_kernel::{Geometry, ops::*};

let cube = Geometry::cube(1.0, 1, 1, 1);
let extruded = poly_extrude(&cube, 0.5, 0.1);
```

### WASM Module

For browsers:

```javascript
const cunning = await import('./cunning_player.js');
await cunning.default();

const cdaBytes = await fetch('asset.cda').then(r => r.arrayBuffer());
cunning.load_cda(new Uint8Array(cdaBytes));
cunning.cook();
```
