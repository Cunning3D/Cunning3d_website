# PolyExtrude (v1.1)

The PolyExtrude node extrudes selected polygons along their normals.

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| distance | float | 1.0 | Extrusion distance |

## Mode

Only **Connected Components** mode available in v1.1.

## Limitations

- No Individual Elements mode (added in v1.2)
- No inset parameter (added in v1.2)
- No divisions parameter (added in v1.2)

# PolyExtrude (v1.1)

PolyExtrude 节点沿法线方向挤出选中的多边形。

## 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| distance | float | 1.0 | 挤出距离 |

## 模式

v1.1 仅支持 **连通分量** 模式。

## 限制

- 无独立元素模式（v1.2 新增）
- 无 inset 参数（v1.2 新增）
- 无 divisions 参数（v1.2 新增）
